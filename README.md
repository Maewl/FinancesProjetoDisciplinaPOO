<h1 align="left"> Projeto Gerenciador de Financiamentos </h1>
<h3 align="left"> Fundamentos da Programação Orientada a Objetos - PUCPR </h3>
<h5 align="left"> :construction: Projeto em construção :construction: </h5>
<p>Aplicação feita em java com intuito de simular financiamentos de imóveis, proposto na disciplina de Fundamentos da Programação Orientada a Objetos.</p>

# Funcionalidades do projeto
- `Consulta de multiplos financiamentos`: Consulta multiplos financiamentos e apresenta seus atributos individuais, além de uma somatória dos valores e financiamentos de todos os imóveis;
- `Cadastro de financiamentos`: Cadastra informações dos financiamentos e calcula valores totais de acordo com os dados informados.
